# ShepherdKids

**Welcome Every Family. Know Every Child. Shepherd Every Journey.**

ShepherdKids is a children's ministry platform built to help church teams care for every child and family with intention.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Deploy

Deploy on [Vercel](https://vercel.com).
